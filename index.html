Telegram Alerts

Speichere Bot API Token und Chat-ID nur lokal auf diesem Gerät. Danach kannst du Alert-Texte kopieren oder direkt an deinen Telegram-Chat senden.

Automatische RSI Alerts Im ZIP ist zusätzlich ein GitHub-Actions-Workflow enthalten. Er prüft alle Seasonality-Assets automatisch auf RSI unter 30 oder über 70 im 4H- und 1D-Chart und sendet dir eine Telegram-Nachricht. Die Bot-Daten für automatische Alerts gehören in GitHub Secrets, nicht sichtbar in den Website-Code.
